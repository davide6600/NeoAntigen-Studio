# Critical Manual Approval Checklist

1. Sequence export for synthesis manifests - Approver: Biosecurity Officer
2. Any wet-lab execution handoff - Approver: Principal Investigator (PI)
3. External data sharing outside project boundary - Approver: Data Manager
4. New patient dataset ingestion with consent mapping - Approver: Data Governance Lead
5. Consent override exception handling - Approver: Ethics/IRB Delegate
6. Promotion of staging model to production - Approver: ML Lead
7. Changes to biosecurity blacklist/motif rules - Approver: Biosecurity Officer
8. Disabling audit logging in any environment - Approver: Security Lead
9. Use of external network data pull mode - Approver: Platform Owner
10. Deletion request execution on patient-linked data - Approver: Privacy Officer
